## Build on

* [Drizzle](https://github.com/trufflesuite/drizzle)
* [React](https://github.com/facebook/react)

## Dependencies for developement

* Truffle 
* Ganache 
* Metamask

### Installing

```
cd client 
npm install

```

## Running the test chain
```
truffle develop 
truffle compile
truffle migrate 
```

## Running the frontend
```
cd client
npm run start

```
You should be good to go.
